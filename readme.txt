== Description ==
This plugin looks for #hashtags in content and creates/links tags.

== Installation ==


1. Upload `link-tag-hashtags.zip` via The Plugins -> Add New screen in the WordPress Dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You should see a new set of wigets in Apperance -> Widgets
